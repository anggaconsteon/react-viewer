🛡️ Patrol sebagai Workflow

Patrol bukan tombol aktivitas biasa — ini adalah guided step-by-step mode dengan state, progress, dan validasi tersendiri.

Start → Scan → Confirm → Complete

🎨 Warna Semantik

Cyan = patrol mode aktif. Hijau = aman/selesai. Merah = insiden/urgensi. Amber = peringatan/masalah. Violet = edge case/offline.

5 warna, 5 makna berbeda

📍 Auto-Capture

Lokasi, timestamp, nama worker, nama checkpoint — semuanya dicatat otomatis. Worker hanya tap, foto, dan konfirmasi status.

Zero typing untuk core actions

🔴 Edge Cases First

QR rusak, offline, patrol terpotong, titik terlewat — semua skenario nyata di lapangan sudah punya UI khusus. Bukan error screen.

Graceful degradation by design

⚡ State Machine

Patrol punya 3 state: Not Started / In Progress / Completed. Setiap state memiliki tampilan berbeda dan aksi yang berbeda.

Worker selalu tahu posisi mereka

📲 Audit Trail

Setiap tap menghasilkan log: checkpoint ID, timestamp, lokasi GPS, foto, status. Data tersedia untuk supervisor secara real-time.

Trackable & auditable by default
