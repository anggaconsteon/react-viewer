🎯 Prinsip Utama

Action-based, bukan menu-based. Setiap screen hanya menampilkan satu tujuan utama dengan tombol besar yang jelas.

Max 2 taps untuk setiap aksi inti

🎨 Warna Semantik

Hijau = aksi utama / aktif. Merah = insiden / urgensi. Amber = tugas / perhatian. Abu-abu = akhiri / tidak aktif.

High contrast untuk outdoor usage

📱 Touch Target

Semua tombol utama minimal 48px tinggi. Chip aktivitas dan tombol insiden dirancang untuk dipencet dengan satu jempol.

Thumb-friendly, satu tangan

🔒 State Management

Sebelum check-in: hanya tampil tombol Mulai Shift. Setelah check-in: tampil 4 aksi. Setelah end shift: semua terkunci.

Zero ambiguity per state

🇮🇩 Microcopy

Bahasa Indonesia informal, singkat, dan tidak teknis. Mengarahkan aksi, bukan menjelaskan fitur.

"Mulai kerja sekarang" bukan "Inisiasi shift"

⚡ Auto-input

Lokasi, timestamp, dan nama worker diisi otomatis. Worker hanya tap — tidak perlu mengetik untuk aksi inti.

Minimal typing, prefer tap + photo
